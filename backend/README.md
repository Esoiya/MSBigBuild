# Flask RESTful service

```Python version: 3.7.5```

Flask app is run directly in the unix server as the root user.
- All of the python modules used need to be installed on the unix server
- For MariaDB connectivity with Python, the Connector plugin also needs to be installed on the server.
- The Public IP address of the server is used as the host name, and the http service is enabled on the server and specified port 8889 in the flask app has been opened on the server.
